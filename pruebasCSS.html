Qué es el CSS:

  • Cascading Style Sheets (hojas de estilo en cascada)
  • Definen cómo se mostrarán los elementos en cada tag de HTML.
  • Sólo existen a partir del HTML 4.0
  • Las hojas de estilo externas nos ahorran mucho trabajo, y están en archivos CSS.

Las instrucciones de CSS se llaman "rules" o reglas. Ejemplo:
h1 {color:blue; font-size:12px;} (eso es una regla).

  • "h1" es el selector de la regla
  • "color:blue" y "font-size:12px" son las declaraciones de la regla
  • "color" y "font-size" son propiedades
  • "blue" y "12px" son valores que les asignamos a las propiedades en las declaraciones.

Cosas especiales para ciegos o gente que tiene problemas para leer, o en el coche cuando conducimos: Aural Style Sheets. Un ejemplo:

Unidades para medir tamaños en CSS:

  • %: Porcentaje
  • in: Pulgada (inch)
  • cm: Centímetro
  • mm: Milímetro
  • em: Tamaño actual de la fuente para el mismo elemento que estamos modificando
  • ex: Como la mitad del tamaño actual de la fuente o así
  • pt: Punto (point) (= 1/72 de pulgada)
  • pc: Pica (= 12 puntos)
  • px: Pixel (punto en la pantalla)

Selector "class": Sirve para definir el estilo de varios elementos que tienen la misma clase. Se usa poniendo algún elemento genérico de HTML seguido de "." (atributo de clase) y la clase. ¡¡LAS CLASES SON LOS ATRIBUTOS DE LOS TAGS EN HTML!! Bueno, no exactamente, más bien son atributos de los tags, sí, pero que al ponerle un valor a un tag en su atributo "class" estamos haciendo que pertenezca a esa clase.

NO OLVIDAR QUE MEDIANTE EL TAG "style" se pueden definir hojas de estilo internas (dentro del tag "head":
<style>
hr {color:blue;}
body {background-color:yellow;}
p {margin-left:20px;}
</style>

Y recordemos que también se hacen en línea con el atributo style="...".

Siempre tienen preferencia los valores que se le den a las propiedades en línea, luego en el CSS interno y luego en el externo. Pero las propiedades no definidas en el tipo de CSS más prioritario se tomarán del siguiente. Prioridades:

  1. Estilo definido en línea
  2. CSS interno (código CSS definido en el tag "style" de "head"
  3. CSS externo (archivo CSS)
  4. Estilo por defecto en el navegador.

¡OJO! Si ponemos el tag "link" que importa el CSS externo después del tag "style" que contiene el CSS interno, el externo sobreescribirá al interno!! OJO OJOOOO XD

Las monospace son tal que todos los caracteres tienen el mismo ancho, son letras como de terminales y tal.
